# Learn Algorithms
JavaScriptでいろいろなアルゴリズムを実装していくレポジトリ

# Stack

```
Transpile: balel + webpack
Document: esdoc
Unit test: ava
CI: Circle CI
```

# Usage

 - build
```
 npm run build
```

- test
```
 npm test
```

# モジュールと用途

 - AVAによるテストのカバレッジをHTMLとして出力する
